Race

Method Preconditions Postconditions
Race(List<Horse>, double, String)
  • horses is not null and contains at least one horse.
  • raceLength is a valid positive number (e.g., non-negative).
  • raceSurface is one of "grass", "dirt", or "mud" (Uses HorseRacingHelper constants).
  • Initializes the Race object with provided values.
  • Sets currentHorse to 0.
  • Initializes an empty results list.
getHorses() None Returns the list of horses participating in the race.
numHorses() None Returns the number of horses participating in the race.
getNextHorse() None Returns the next horse in the race rotation.
getRaceLength() None Returns the length of the race in furlongs.
getRaceSurface() None Returns the surface type of the race ("grass", "dirt", or "mud").
displayRaceInfo() None Prints information about the race, including surface, length, and list of horses.
displayResults() None Prints the results of the race, including the order in which horses finished.
startRace() None Simulates the race, updating horse positions until all horses finish. Displays results and stops background music after the race.
resetHorses() None Resets the position of all participating horses and marks them as not finished.

HorseRacingHelper

Method Static Preconditions Postconditions
pauseForMilliseconds(int) Yes
  • milliseconds is a valid positive integer.
  • Pauses the execution of the program for the specified number of milliseconds.
clearConsole() Yes None Clears the console screen.
updateTrack(int, List<Horse>) Yes
  • numSpaces is a valid positive integer.
  • horses is not null.
  • Updates the track display with horse positions based on the number of spaces and list of horses.
playBackgroundMusic(String, boolean) Yes
  • filename is a valid non-empty string representing the music file path.
  • loop is a valid boolean value.
  • Plays background music with the specified file name, optionally in a loop.
playBackgroundMusicAndWait(String) Yes
  • filename is a valid non-empty string representing the music file path.
  • Plays background music with the specified file name and waits for it to finish.
stopMusic() Yes None Stops the currently playing background music.
